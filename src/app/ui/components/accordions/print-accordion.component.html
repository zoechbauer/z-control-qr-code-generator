Note

Select the QR code size depending on the amount of text entered. The more text, the larger the QR code should be so that your smartphone’s camera app can quickly scan the QR code. Special QR code scanner apps can also recognize smaller QR codes.

The number of QR codes per A4 page depends on the selected size, the gap between the QR codes, and the amount of text entered.

You can set the number in the “Number of QR codes per A4 page” selection. The “As many as possible” option calculates the maximum number per page based on size and gap. The “I want to enter the number” option lets you set a lower number.

The result of your selection is shown in the result line and in the settings panel in the format [ Size / Gap / Number ] or [Size / 1].

After sending the email, you can change the settings here and resend the email.

For more information, see the help page in the chapter “Print QR code in a specific size”.

Hinweis

Wählen Sie die Größe des QR-Codes abhängig von der Menge des eingegebenen Textes. Je mehr Text, desto größer sollte der QR-Code sein, damit die Kamera-App Ihres Smartphones den QR-Code schnell scannen kann. Spezielle QR-Code-Scanner-Apps können auch kleinere QR-Codes erkennen.

Die Anzahl der QR-Codes pro A4-Seite hängt von der gewählten Größe, dem Abstand zwischen den QR Codes und der erfassten Textmenge ab.

Sie können die Anzahl in der Auswahl „Anzahl QR-Codes pro A4-Seite“ festlegen. Mit der Option „So viele wie möglich“ wird die maximale Anzahl pro Seite basierend auf Größe und Abstand berechnet. Mit der Option „Ich möchte die Anzahl eingeben“ können Sie eine geringere Anzahl festlegen.

Das Ergebnis Ihrer Auswahl wird in der Ergebniszeile und im Einstellungs-Panel im Format [ Größe / Abstand / Anzahl ] oder [ Größe / 1] angezeigt.

Nach dem Versenden der E-Mail können Sie die Einstellungen hier ändern und die E-Mail erneut versenden.

Weitere Informationen finden Sie auf der Hilfeseite im Kapitel „QR-Code in einer bestimmten Größe drucken“.
