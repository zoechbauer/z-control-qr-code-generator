Was kann die App?

Es wird aus dem erfassten Text ein QR-Code erstellt und per Mail versendet.

Dieser QR Code wird als Bild und PDF dem Mail als Anhang mitgegeben und der erfasste Text wird im Mail angeführt. Dadurch sieht man sofort welcher Text in den QR Codes enthalten ist.

Damit beim Mailversand nicht immer die Mailadresse erfasst werden muss, können Mail-Adressen verwaltet werden.

Wie bediene ich die App?

Überblick
  1. Text erfassen
  2. QR Code generieren
  3. QR Code mailen
  4. Optional Mailadresse(n) erfassen
Detail-Informationen QR Code generieren
  1. Ein- oder mehrzeilige Texte erfassen.
    Eine neue Zeile oder Leerzeilen können durch ein- oder mehrmaliges Drücken der Eingabetaste eingefügt werden.

  2. Schaltfläche generieren drücken.
    Es wird dann der genertierte QR-Code angezeigt.
    Neben dem Label QR Code Text wird die Textlänge und die maximale Textlänge folgendermaßen angezeigt: [erfasste Textlänge/maximale Textlänge]

  3. Mittels Schaltfläche QR Code speichern und mailen werde die QR Codes dem Mail angehängt.

    Das Mail kann abgeändert werden und wird mit dem Pfeil rechts in der rechten Kopfzeile versendet.

      Inhalt der Mail
    • Betreff: QR Code mit Angabe der Textlänge des erfassten Textes.
      z.B. QR Code mit Textlänge 512

    • Text: Der in den QR Codes enthaltene erfasste Text.

    • Empfänger: Der oder die Empfänger werden automatisch vorbelegt, wenn sie mittels MAIL ADR angelegt wurden. Sonst müssen sie erfasst werden.

  4. Text ändern oder löschen.
    Der Text kann jederzeit geändert werden, es muss aber nach einer Textänderung wieder die Schaltfläche generieren gedrückt werden.

    Der Text kann mittels Schaltfläche Eingabe Text löschen gelöscht werden.

Detail-Informationen Mail Adressen verwalten

In der Kopfzeile gibt es die Auswahl MAIL ADR

Hier kann zwischen QR-Code Erstellung und Mail-Adressenverwaltung gewechselt werden.

Es werden alle bereits erfassten Mail-Adressen angezeigt und neue Mail-Adressen können erfasst und bereits erfasste Mail-Adressen können gelöscht werden.

Technische Informationen

Die erfassten Mailadressen werden am Handy in local Storage (IndexedDB bzw. Local Storage) gespeichert.

Der generierte QR Code wird als Bild (PNG-Format) und PDF am Handy im {{ folderName }} als {{ fileNamePng }} und als {{ fileNamePdf }} gespeichert.
