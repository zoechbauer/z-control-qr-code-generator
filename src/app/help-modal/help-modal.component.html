Was kann die App?

Es wird aus dem erfassten Text ein QR Code erstellt und per Mail versendet. Der Text kann nicht formatiert werden und kann bis zu {{maxInputLength}} Zeichen lang sein.

Der QR Code wird als Bild und PDF dem Mail angehängt und der erfasste Text steht im Mail. So sehen Sie sofort welcher Text im QR Code enthalten ist, falls Sie mehrere QR Codes erstellen.

Damit Sie beim Mailversand nicht immer die Mailadresse erfassen müssen, können Sie Mail-Adressen anlegen.

Wie bediene ich die App?

Überblick
  1. Text erfassen
  2. QR Code generieren
  3. QR Code mailen
  4. Optional Mailadresse(n) erfassen
Detail-Informationen QR Code generieren
  1. Ein- oder mehrzeilige Texte erfassen.
    Eine neue Zeile oder Leerzeilen können durch ein- oder mehrmaliges Drücken der Eingabetaste eingefügt werden. Als Formatierung wird nur die Großschreibung unterstützt. Rechts unterhalb dem Eingabefeld werden die erfasste Textlänge und die maximale Textlänge von {{maxInputLength}} Zeichen angezeigt.

  2. Schaltfläche generieren drücken.
    Es wird dann der genertierte QR Code angezeigt.

  3. Mittels Schaltfläche QR Code mailen werden die QR Codes dem Mail als png und pdf angehängt.

    Das Mail kann bei Bedarf bearbeitet werden und wird mit dem Pfeil rechts in der rechten Kopfzeile versendet.

      Inhalt der Mail
    • Betreff: QR Code mit Angabe der Textlänge des erfassten Textes.
      z.B. QR Code mit Textlänge 512

    • Text: Der in den QR Codes enthaltene erfasste Text.

    • Empfänger: Der oder die Empfänger werden automatisch vorbelegt, wenn sie mittels MAIL ADR angelegt wurden. Sonst müssen sie erfasst werden.

  4. Text ändern oder löschen.
    Der Text kann jederzeit geändert werden, es muss aber nach einer Textänderung wieder die Schaltfläche generieren gedrückt werden.

    Der Text kann mittels Schaltfläche Eingabe Text löschen gelöscht werden.

Detail-Informationen Mail Adressen verwalten

In der Kopfzeile gibt es die Auswahl MAIL ADR

Hier kann zwischen QR Code Erstellung und Mail-Adressenverwaltung gewechselt werden.

Es werden alle bereits erfassten Mail-Adressen angezeigt und neue Mail-Adressen können erfasst und bereits erfasste Mail-Adressen können gelöscht werden.

Technische Informationen

Spracheingabe: Wenn Sie in den Handy-Einstellungen die Spracheingabe aktivieren, erscheint im Tastaturfeld ein Mikrofon-Symbol. Durch Tippen auf dieses Symbol wechseln Sie zur Spracheingabe. Sie können flexibel zwischen Tastatur und Spracheingabe wechseln. Die Bedienung entspricht der üblichen Spracheingabe-Funktion Ihres Handys.

Scannen des generierten QR Codes: Der erfasste Text kann bis zu {{maxInputLength}} Zeichen lang sein. Beachten Sie: Je mehr Text der QR Code enthält, desto länger dauert der Scan-Vorgang. Für sehr lange Texte empfehlen wir eine spezielle Scanner-App. QR Codes mit bis zu 500 Zeichen können in der Regel problemlos mit der Kamera-App Ihres Handys gescannt werden. Bei den meisten modernen Smartphones ist die QR Code-Scan-Funktion bereits in der Standard-Kamera-App integriert.

Erfasste Mailadressen werden am Handy in local Storage (IndexedDB bzw. Local Storage) gespeichert.

Der generierte QR Code wird am Handy im {{ folderName }} als {{ fileNamePng }} und {{ fileNamePdf }} gespeichert.
