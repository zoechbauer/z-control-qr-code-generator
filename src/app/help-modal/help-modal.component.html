Was kann die App?

Es wird aus dem erfassten Text ein QR-Code erstellt.

Dieser QR Code kann als Bild (PNG-Format) und/oder als PDF am Handy gespeichert werden und wird im {{ folderName }} als {{ fileName }} abgelegt.

Wie bediene ich die App?

  1. ein- oder mehrzeilige Texte erfassen.
    Eine neue Zeile oder Leerzeilen können durch Drücken der Eingabetaste eingefügt werden.

  2. Schaltfläche generieren drücken.
    Es wird dann der genertierte QR-Code angezeigt.
    Neben dem Label QR Code Text wird die Textlänge und die maximale Textlänge folgendermaßen angezeigt: [erfasste Textlänge/maximale Textlänge]

  3. Soll der QR Code als Bild gespeichert werden, dann die Schaltfläche QR PNG drücken.
    Es wird dann eine Nachricht angezeigt, wo die Datei gespeichert wurde. Bitte auf OK klicken.

  4. Soll der QR Code als PDF gespeichert werden, dann die Schaltfläche QR PDF drücken.
    Es wird dann eine Nachricht angezeigt, wo die Datei gespeichert wurde. Bitte auf OK klicken.

  5. Text ändern oder löschen.
    Der Text kann jederzeit geändert werden, es muss aber nach einer Textänderung wieder die Schaltfläche generieren gedrückt werden.
    Der Text kann mittels Schaltfläche QR löschen gelöscht werden.
